# AI Skin Disease Scanner

AI Skin Disease Scanner is a web-based healthcare application that helps users identify potential skin conditions using artificial intelligence and computer vision. The system allows users to scan their skin through a camera or upload an image, and a trained machine learning model analyzes the image to predict possible skin diseases.

The goal of this project is to provide quick and accessible preliminary skin health insights, especially for people who may not have immediate access to dermatologists.

---

## Features

- AI-based skin disease detection
- Real-time skin scanning using camera
- Image upload for skin analysis
- Basic remedies and skincare suggestions
- AI chatbot for dermatology-related questions
- Simple and user-friendly web interface

## How It Works

1. The user opens the web application.
2. The user scans their skin using the camera or uploads an image.
3. The system processes the image using OpenCV.
4. A trained machine learning model analyzes the skin features.
5. The model predicts the possible skin condition.
6. The user receives basic information and suggested remedies.
